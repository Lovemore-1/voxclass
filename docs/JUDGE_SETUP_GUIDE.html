Shortcut Asia 2026

This guide gets VoxClass running on your machine in under 10 minutes. The app needs three things: Flutter (already on most dev machines), a Supabase project (free), and a Gemini API key (free). Everything else is already in the repo.

1
Get the Code
Repo
github.com/munet/voxclass
Requires
Flutter SDK 3.x — verify with flutter --version
git clone https://github.com/munet/voxclass.git
cd voxclass
flutter pub get
2
Create a Supabase Project (free)
  1. Go to supabase.com → New Project (takes ~1 min to provision)
  2. In the left sidebar → SQL Editor → paste the entire contents of supabase/schema.sql from the repo → click Run
  3. Go to Table Editor → open each of these tables and enable Realtime: reactions, ai_questions, slides
  4. Go to Storage → New Bucket → name it slides → set to Public
  5. Go to Project Settings → API → copy your Project URL and anon public key
3
Get a Gemini API Key (free)
  1. Go to aistudio.google.com/apikey
  2. Click Create API Key → copy it
4
Configure Environment

In the project root, copy .env.example to .env and fill in your three keys:

# .env
SUPABASE_URL=https://your-project-id.supabase.co
SUPABASE_ANON_KEY=eyJ...
GEMINI_API_KEY=AIza...
Note: The .env file is gitignored and never committed — your keys stay private.
5
Run the App
# Recommended — runs in Chrome, zero extra setup
flutter run -d chrome

# OR — Windows desktop app
flutter run -d windows
Windows desktop only: Requires Developer Mode enabled.
Settings → System → For Developers → toggle Developer Mode ON.
6
Test the Full Flow
Lecturer
Sign up → choose Lecturer → Dashboard → Start Class → give a title → session starts with a live 6-digit code
Student
Sign up → choose Student → Dashboard → Join Class → enter the 6-digit code → react live
AI Demo
In the live session → Slides tab → upload any JPG → tap Ask Gemini → 3 quiz questions appear instantly
Polish Mode
Dashboard → Polish Mode → paste any text → pick a style → see AI rewrite with word-level diff
